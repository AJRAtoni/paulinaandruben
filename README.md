# Paulina & Ruben Wedding Website

Sitio web oficial para la boda de Paulina Cuevas y Ruben Moreno.

**Fecha:** 6 de junio de 2026
**Lugar:** Tánger, Marruecos

---

## 🌍 Ver el sitio

El sitio está desplegado y disponible en: [https://paulinaandruben.github.io](https://paulinaandruben.github.io)

---

## 📋 Secciones

- **Home**: Información principal, fecha, lugar y cuenta regresiva
- **Itinerario**: Agenda completa de eventos para el fin de semana
- **Alojamiento**: Recomendaciones de hoteles
- **RSVP**: Formulario para confirmar asistencia
- **FAQs**: Preguntas frecuentes sobre el viaje y la boda

---

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)


## 📝 Características

- Diseño responsive (móvil y escritorio)
- Animaciones suaves con CSS
- Menú de navegación móvil
- Formulario de RSVP integrado con Google Sheets
- Temporizador de cuenta regresiva en tiempo real
- FAQ acordeón interactivo
- Integración con Google Maps para ubicaciones (botones extraíbles)
